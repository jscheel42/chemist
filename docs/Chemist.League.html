    challenger(region, opts \\ %{})

    Contains league entries for the challenger league.

    Default opts:

    • type: “RANKED_FLEX_SR”

      • Type of matchmaking

        • RANKED_FLEX_TT
        • RANKED_SOLO_5x5

        • RANKED_TEAM_5x5
