    ranked(region, summoner_id, opts \\ %{})

    Contains ranked stats information; retrieved by summoner id.

    Default opts:

    • season: defaults to current season

      • Specifies ranked season from which to gather data.

      • Possible values:

        • SEASON3
        • SEASON2014
        • SEASON2015
        • SEASON2016
        • SEASON2017
